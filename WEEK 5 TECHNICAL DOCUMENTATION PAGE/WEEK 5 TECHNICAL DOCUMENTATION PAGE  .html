Introduction

JavaScript is a cross-platform, object-oriented scripting language used to make webpages interactive.

It contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements.

console.log("Welcome to JS");
Hello World

To get started with writing JavaScript, the classic entry point is printing a hello message.

This is commonly executed inside your browser's console or linked directly through your web document headers.

alert("Hello World");
  • Console logs
  • Alert popups
Variables

You use variables as symbolic names for values in your application. The names of variables are called identifiers.

A JavaScript identifier must start with a letter, underscore, or dollar sign.

let myVariable = 42;
Data Types

The latest ECMAScript standard defines eight structural primitive and object data types.

JavaScript is a dynamically typed language, meaning you do not have to declare the type of a variable before it is used.

typeof "Hello"
  • String
  • Number
  • Boolean
Functions

A function definition consists of the function keyword, followed by the name of the function, and a list of arguments.

The statements inside the block determine the underlying code execution pipeline whenever the function name is called.

function greet() { return "Hi!"; }
